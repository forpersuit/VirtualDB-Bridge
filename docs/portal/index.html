Transparent Proxy

Applications talk to the Bridge using standard MySQL protocol. No proprietary libraries or complex SDKs needed.

透明代理

应用程序使用标准 SQL 与网关通信。无需任何私有库或复杂的 SDK。

Dynamic Isolation

Every query is inspected and rewritten to include tenant-specific prefixes, ensuring physical data separation.

动态隔离

每一条查询都会被解析并重写，自动注入租户专属前缀，确保物理层面的数据隔离。

Full API Specification 完整 API 技术规范

Global Authentication Headers

全局身份验证请求头

Header Type Description
CF-Access-Client-Id String Cloudflare Zero Trust Service Token ID. Cloudflare Zero Trust 服务令牌 ID。
CF-Access-Client-Secret String Cloudflare Zero Trust Service Token Secret. Cloudflare Zero Trust 服务令牌密钥。
POST /query

Executes virtualized SQL queries against the multi-tenant database cluster.

在多租户数据库集群上执行虚拟化 SQL 查询。

Request Body (JSON)

Field Type Required Description
apiKey String Yes Tenant identification key from tenants.json. 来自于 tenants.json 的租户唯一识别密钥。
sql String Yes Standard MySQL query (rewrite engine will auto-prefix). 标准 MySQL 语句（重写引擎会自动注入前缀）。
params Array No Values for parameterized queries (prevent SQL injection). 参数化查询的值数组（用于防范 SQL 注入）。
POST /email/send

Sends verified emails via SMTP proxy with tenant-level authorization.

通过 SMTP 代理发送经过租户权限校验的电子邮件。

Request Body (JSON)

Field Type Required Description
to String|Array Yes Recipient email address(es). 收件人地址或地址数组。
subject String Yes Email subject line. 邮件主题。
html | text String Yes Body of the email (must provide at least one). 邮件正文（至少提供 HTML 或纯文本之一）。
from String No Verified sender email (must exist in vmail.mailbox). 经验证的发件人地址（必须存在于 vmail.mailbox 中）。
fromName String No Display name for the sender. 发件人显示的友好名称。

Zero Trust Defense Matrix 零信任防御矩阵

Developer Error

Forgot a WHERE clause? In normal shared DBs, this leaks ALL tenant data.

开发者疏忽

漏写了 WHERE 过滤条件？在普通共享库中，这会导致全库数据泄露。

Defense

SQL Virtualization forces isolation at the network layer. Illegal queries are rewritten before execution.

SQL 虚拟化在网络层强制隔离。非法查询在执行前即被重写修正。

Infrastructure Attack

Hackers attempt to connect directly to your database IP using brute force.

基础设施攻击

黑客尝试通过暴力破解直接连接您的数据库物理 IP。

Defense

Zero Trust Perimeter. DB only talks to the Bridge internal IP. Origin IP is hidden via Cloudflare.

零信任边界。数据库仅与网关内网通信。真实 IP 已通过 Cloudflare 彻底隐藏。

Compromised Logic

Even if an attacker hijacks the Gateway code, they attempt cross-prefix access.

逻辑层沦陷

即便攻击者劫持了网关代码并尝试发起跨前缀访问。

Defense

DB Wildcard Permissions. The DB engine rejects the request as the user only owns tenantA_%.

数据库通配符权限。数据库引擎会直接拒绝，因为该账号物理上仅拥有 tenantA_% 权限。
